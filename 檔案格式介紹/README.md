# 專有名詞介紹
`.yml` 檔案的組成皆為「屬性: 值」，例如「price: 50」。以下提供常見的屬性名字與其中文翻譯。

## 屬性對照表
| 屬性         | 翻譯 |
|--------------|----|
| price        |價格|
| build-reps   |建築需求|
| build-radius |建築範圍|
| effects      |對非成員玩家的限制|
| period       |作用週期|
| upkeep       |作用|
| reagents     |需要放入箱子內，但不會消耗的資源|
| input        |需要放入箱子內，會消耗的資源|
| output       |產生於箱子內的資源|
| payout       |為建造者賺的錢|
